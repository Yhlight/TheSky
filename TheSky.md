## 天空
天空是一款基于HTML + CSS + TS + Vue + **C++ + WASM(可选，优化画面绘制的性能)**的一款专注于画面演出的2D游戏  

玩家需要操控手中的晶体向右移动，欣赏内置或程序化生成的无缝衔接的画面  
体验春天的草地，夏天的太阳，秋天的风与落叶，冬天的雪地与飘雪  
甚至是星空，海洋，乡村，沙漠，等几十种画面，所有的画面或许内置或许程序化生成  

玩家可以使用按键D向右进行移动，长按三秒后固定移动，也可以长按键盘SPACE键(三秒后固定)  
此时的晶体会向右飞行，拽出拖尾，并且画面加速，展现出时间加速的效果，快速的无缝场景变换
