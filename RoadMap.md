# TheSky Development Roadmap

This document outlines the development progress of TheSky, a 2D web game.

## Version 1.0 (Completed)

- **Core Gameplay:** Implemented the basic mechanics of controlling a crystal that moves across the screen.
- **Scene System:** Created a system for transitioning between different scenes.
- **Visuals:** Added four seasonal scenes (Spring, Summer, Autumn, Winter) with unique background colors and particle effects.
- **Controls:** Implemented keyboard controls for movement, including a 3-second hold to engage auto-movement.

## Version 1.1 (Completed)

- **Enhanced Visuals:** Refactored the scene management system to support more visually distinct and immersive environments.
- **New Scenes:** Added "Starry Night" and "Desert" scenes with unique particle effects and styling.
- **Time Acceleration:** Implemented an accelerated movement effect when auto-move is active.

## Future Goals

- **Procedural Scene Generation:** Introduce procedurally generated scenes to create a more dynamic and unpredictable experience.
- **Sound Design:** Add a calming and atmospheric soundtrack and sound effects to enhance the game's immersive quality.
- **Performance Optimization:** Explore using C++ and WASM to optimize performance for more complex visual effects.
- **Expanded Content:** Create a wider variety of scenes, particle effects, and interactive elements to discover.
